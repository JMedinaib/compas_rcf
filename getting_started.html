Getting started

Installation

Note

All code related to fabrication with ABB industrial robots requires compas_rrc which is hosted in a private repository.

  1. (Optional) Create a virtual environment using your tool of choice (e.g. virtualenv, conda, etc).

    • Using Anaconda

    conda config --add channels conda-forge
# use conda to install compas_fab if possible
conda create -n compas_rcf python=3.7 git compas_fab
conda activate compas_rcf

    • Using virtualenv

    virtualenv --python=python3.7 {{path/to/venv}}
source {{path/to/venv}}/bin/activate

  2. Install compas_rrc & compas_rcf

    # install compas_rrc separately
pip install git+https://bitbucket.org/ethrfl/compas_rrc#egg=compas_rrc-v0.2.2
pip install compas_rcf
# or compas_rcf=={version} for specific version

    If you have authentication issues with BitBucket (for compas_rrc), see Authentication problems with BitBucket

  3. (Optional) Make package accessible in Rhino and Grasshopper

    python -m compas_rcf.install_rhino

Update

To update the repository run:

pip install -U compas_rcf
